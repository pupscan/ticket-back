# ticket-back


## Set your workspace

To run the application you need to set variable environements:

- FACEBOOK_APP_ID
- FACEBOOK_APP_SECRET
- TWITTER_APP_ID
- TWITTER_APP_SECRET
- TWITTER_TOKEN_ID
- TWITTER_TOKEN_SECRET
- INDIEGOGO_APP_TOKEN

To set on Mac OSX run the command `launchctl setenv FACEBOOK_APP_SECRET "your_facebook_secret"`
